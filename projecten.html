Cicero Bot

Een discord bot die gebruik maakt van de discord.py API om asynchrone requests af te handelen. De applicatie word gebruikt in een discord server van 100 gebruikers waarvan ~30 de applicatie dagelijks geruiken.

Aan de hand van een .txt database heeft de bot enkele functionaliteiten zoals een Issue Tracking System (ITS), een anoniem stemsysteem dat via emojis werkt en een piechart kan tonen op basis van stemresultaten alsook immuun te zijn tegen VPN, omzetten van eenheden en tijdzones, en wekelijkse notificaties.

Deze bot word lokaal op mijn Raspberry pi gehost en gebruikt jenkins' continuous deployment met github; Elke 6 uur word github code opgehaald en word de bot opnieuw opgestart.
Deze applicatie zal nooit definitief afzijn en ik werk er weekelijks aan verder.
In de toekomst heb ik plannen om van mijn .txt database af te stappen en een SQL database te gebruiken via MariaDB.
