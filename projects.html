Cicero Bot

A discord bot that makes use of the discord.py API to handle assynchronous requests. The application is used in a discord server of roughly 100 users with roughly 30 using it on a daily basis.

Aan de hand van een .txt database heeft de bot enkele functionaliteiten zoals Through the use of a .txt database the bot has a few functionalities: Issue Tracking System (ITS), an anonymous voting system that works through emojis and makes use of pie charts to show results. It has gone through many iterations as issues came along such as the use of a VPN to cast multiple votes converting of units and timezones, weekly notifications

The bot is hosted locally on a Raspberry pi and utilises jenkins for continuous deployment with Github; Every update to the bot is immediately deployed.
This application will never be completely finished and I still work on it regularly
I have future plans to update the database to a SQL database to avoid struggles that come along with .txt files
